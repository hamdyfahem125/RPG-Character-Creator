# RPG Character Creator

A small Python application to create and display characters for an RPG (Role-Playing Game).  
This project helps practice Python basics such as functions, validation, string formatting, and loops.

---

## Features

- **Validate Character Name:**
  - Must be a string
  - Cannot be empty
  - Maximum 10 characters
  - Cannot contain spaces
- **Validate Character Stats (Strength, Intelligence, Charisma):**
  - Must be integers
  - Values must be between 1 and 4
  - Total points must equal 7
- **Display Character Stats Visually:**
  - `●` for full points
  - `○` for empty points
  - Each stat is displayed out of 10 dots
- **Easily extendable** for new stats or different point systems
